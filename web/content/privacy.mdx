# Privacy Policy

*Last updated: March 2026*

---

## 1. Overview

Yu-Jin is a Discord bot. This policy explains what data is collected, how it is used, and your rights regarding that data.

---

## 2. Data Collected

Yu-Jin stores the following data in a configuration file (`guild-config.json`) on the server host:

| Data | Why it's stored |
|---|---|
| Discord guild (server) ID | Identifies which server a config belongs to |
| Discord user IDs | Used for mention settings, favor ledger entries, and absence watches |
| Voice join timestamps | Used for absence detection (when a user last joined a voice channel) |
| Favor records | Stores the from-user, to-user, reason, timestamp, and settled status of each favor |
| Absence watch settings | Stores the threshold (days) and last-alerted timestamp per watched user |

Yu-Jin does **not** collect:
- Message content
- Real names or email addresses
- Any data beyond what is listed above

---

## 3. How Data Is Used

All collected data is used solely to operate the bot's features within your Discord server. Data is not sold, shared, or transmitted to any third parties.

---

## 4. Data Storage

Data is stored in a plain JSON file on the server hosting the bot. It is not stored in any cloud database or third-party service controlled by the bot developers.

---

## 5. Data Retention

Data persists until it is manually deleted from the configuration file or the bot is removed from the server. There is no automated data deletion.

---

## 6. Your Rights

If you are a member of a server running Yu-Jin and would like your data removed (e.g., favor records, absence watch entries), contact your server administrator. Server administrators can edit or delete configuration data directly.

---

## 7. Changes to This Policy

This policy may be updated at any time. The "last updated" date at the top of this page reflects the most recent revision.

---

## 8. Contact

For questions about this policy, open an issue on the <a href="https://github.com/joyousdevil/yu-jin-discord" target="_blank" rel="noreferrer">GitHub repository</a> or contact the bot's operator directly.
